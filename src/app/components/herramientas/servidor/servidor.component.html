Modo Servidor

Activa la opcion modo servidor para compartir la ip de tu maquina servidor en el ambiente de red, de esta manera puedes tener más de una maquina trabajando.

Puntos a considerar:
► Al activar o desactivar la función el servidor se reiniciara, tranquilo, es normal.
► Es importante que configures el router para que la máquina servidor tenga IP fija.
► Si el modo de red falla por alguna razón, siempre puedes continuar en tu máquina servidor.
► Para que las otras terminales puedan funcionar, es necesario que la máquina servidor este prendida y conectada a la red.
► El modo servidor está disponible con la mensualidad, disfrutalo.
